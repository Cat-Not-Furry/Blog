Exportar a JSON

1) Haz clic en Descargar projects.json para guardar el archivo.
2) En la raíz del proyecto, ejecuta el script pasando la ruta al archivo descargado:

node scripts/update-projects.js ~/Downloads/projects.json

El script copia el JSON a data/projects.json. Ajusta la ruta si descargaste en otra ubicación.
